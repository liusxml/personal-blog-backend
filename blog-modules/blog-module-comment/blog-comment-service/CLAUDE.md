# blog-comment-service 模块

## 模块职责
评论模块业务逻辑实现，负责评论管理相关的业务逻辑、数据持久化、事务管理等服务层实现。

## 入口与启动
- 此模块为服务实现模块，不包含启动类
- 实现blog-comment-api模块定义的接口
- 需要配置数据源和事务管理

## 对外接口
- 实现评论相关的所有业务服务接口
- 提供评论CRUD操作服务
- 暴露服务接口供控制器调用

## 关键依赖与配置

### 主要依赖
- MyBatis-Plus 3.5.6 - 数据访问
- Spring Data JDBC/Transaction - 事务管理
- 数据库连接池（HikariCP）
